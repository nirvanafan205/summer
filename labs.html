Form Validation

Purpose: You are going to learn basic form validation and event handling in JavaScript. Includes 5 different input boxes...

Username (a text field) The user should enter only lower-case letters in this field, or the entry is invalid. Checks this while the user is typing or when the field is blurred.

Password (a password field) The user should enter a password containing a mix of upper-case and lower- case letters and numbers (at least one of each, or the entry is invalid). Other characters are allowed but optional. Checks this while the user is typing or when the field is blurred

Student ID number (a text field) The user should enter a 9 digit number. No letters are allowed or the entry is invalid. The number must be exactly 9 digits long to be valid.

Message (a text area) The user may enter up to 25 words. The field should be disabled if 25 words are entered, such that no other words can be typed into the textarea. This can be accomplished a few different ways. A text label next to the Message field should contain a number which counts down from 25 to 0 as the user enters words.

Submit (a button input type, or a submit input type if you can get this to work without actually submitting the form)
